# 个人导航博客

一个兼具导航功能和个性化内容展示的响应式个人博客网站，支持手机和电脑双端访问。

## 🚀 项目特点

- **响应式设计**：完美适配手机和电脑设备
- **分类导航**：按个人、技术、开发、知识、娱乐、游戏等分类组织链接
- **主题切换**：支持亮色/暗色模式，可根据系统偏好自动切换
- **个性化内容**：包含个人信息、技能展示和社交媒体链接
- **动画效果**：使用Framer Motion添加流畅的过渡动画
- **模块化结构**：代码组织清晰，便于维护和扩展

## 🔧 技术栈

- **前端框架**：React 18+
- **编程语言**：TypeScript
- **构建工具**：Vite
- **样式框架**：Tailwind CSS
- **动画库**：Framer Motion
- **图标库**：Font Awesome

## ⚙️ 环境配置

### Node.js 版本建议

为了确保项目能正常运行，建议下载并安装 **Node.js 最新的 LTS（长期支持）版本**。LTS版本提供更好的稳定性和更长时间的官方支持，非常适合开发环境使用。

### 安装步骤

1. 从 [Node.js 官网](https://nodejs.org/) 下载并安装最新的 LTS 版本
2. 验证安装是否成功：
   ```bash
   node -v  # 显示Node.js版本
   npm -v   # 显示npm版本
   ```
3. 克隆或下载本项目
4. 进入项目目录，安装依赖：
   ```bash
   npm install
   # 或使用pnpm
   pnpm install
   ```
5. 启动开发服务器：
   ```bash
   npm run dev
   # 或
   pnpm dev
   ```
6. 构建生产版本：
   ```bash
   npm run build
   # 或
   pnpm build
   ```

## 📝 本地化修改指南

### 个人信息修改

在 `src/pages/Home.tsx` 文件中，您可以修改以下内容：

- **个人基本信息**：包括姓名、简介、头像和背景图片
- **社交媒体链接**：添加或修改社交媒体图标和链接
- **技能标签**：自定义展示的技能和兴趣

### 导航链接修改

同样在 `src/pages/Home.tsx` 文件中：

- **添加新链接**：在 `navigationLinks` 数组中添加新的链接对象
- **修改分类**：调整 `categories` 数组来自定义分类名称和图标
- **调整顺序**：通过修改数组中元素的顺序来调整显示顺序

### 界面样式调整

- **博客标题和图标**：在 `src/components/BlogHeader.tsx` 文件中修改
- **页脚信息**：在 `src/components/Footer.tsx` 文件中自定义页脚文本和链接
- **卡片样式**：在 `src/components/LinkCard.tsx` 文件中调整卡片的外观

## 📚 功能说明

### 分类筛选

点击页面顶部的分类按钮，可以快速筛选出特定类型的链接。点击"全部"按钮可以查看所有链接。

### 主题切换

通过页面右上角的太阳/月亮图标，可以手动切换亮色/暗色模式。系统也会根据您的设备偏好自动选择合适的主题。

### 链接导航

所有链接卡片都支持点击跳转到对应的网站，点击时会在新标签页中打开，确保您的博客页面不会被关闭。
